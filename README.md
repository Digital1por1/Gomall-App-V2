# GOMALL | Creator Studio

Esta es una herramienta profesional para la creación de contenido publicitario impulsada por IA, diseñada específicamente para socios de **GOMALL**.

## 🚀 Características
- **IA Generativa:** Crea fondos de alta calidad con Gemini 2.5 Flash.
- **Edición en Tiempo Real:** Arrastra y suelta textos y logos directamente sobre el canvas.
- **Formatos Automáticos:** Exporta diseños optimizados para Feed (3:4) y Stories (9:16).

## 🛠️ Instalación y Desarrollo

1. Clona el repositorio:
   ```bash
   git clone https://github.com/Digital1por1/-Digital1por1-Gomall-Creator-Studio-.git
   ```
2. Instala las dependencias:
   ```bash
   npm install
   ```
3. Ejecuta el proyecto en local:
   ```bash
   npm run dev
   ```

## 📦 Despliegue
Este proyecto está optimizado para ser desplegado en **Vercel** o **Netlify**. Solo tienes que conectar tu repositorio de GitHub y se detectará automáticamente la configuración de Vite.
