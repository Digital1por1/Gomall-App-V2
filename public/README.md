# Archivos públicos

Poné acá la imagen del hero de la landing con el nombre exacto **`hero.jpg`**:

```
public/hero.jpg
```

- Recomendado: la foto en formato **4:5** (vertical tipo post de feed).
- Si el archivo no existe, la landing muestra un fondo degradado de respaldo (no se rompe).
- Tras agregarla, hacé commit/push y redeploy: queda en `/hero.jpg`.
